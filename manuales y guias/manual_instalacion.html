Manual de Instalación Concrete Jungle

Requisitos del Servidor

  • Sistema Operativo: Linux
  • Servidor web: Apache versión 2.4.62 o superior
  • PHP versión 8.4
  • Servidor de base de datos: MariadDB version 11.5
  • Servidor FTP que soporte conexiones seguras (SFTP) con FileZilla

Requisitos del Cliente

  • Navegador web: Google Chrome, Mozilla Firefox, Microsoft Edge o Safari (versiones actualizadas)
  • Conexión a Internet estable

Requisitos para el desarrollador

  • Editor de código: Visual Studio Code o cualquier otro editor de su preferencia
  • Control de versiones: Git
  • Entorno de desarrollo local: XAMPP, WAMP o MAMP
  • Cliente FTP FileZilla
